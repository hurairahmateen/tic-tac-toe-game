# Tic Tac Toe Game

A clean, minimalist Tic Tac Toe game built with React and Vite.

## Features

- 2-player local gameplay (X vs O)
- Win detection with highlighted winning line
- Draw detection
- One-click reset
- Polished UI inspired by a soft, minimal design

## Tech Stack

- React
- Vite
- CSS

## Getting Started

### Install

```bash
npm install
```

### Run (Dev)

```bash
npm run dev
```

Open the local URL shown in the terminal to play.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

- `src/App.jsx` — game logic and UI
- `src/App.css` — game styles
- `src/index.css` — global styles
- `index.html` — app shell and metadata
